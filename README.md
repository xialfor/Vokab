# Vokab

Lightweight vocabulary trainer to help learn and review words efficiently.

## Features
- Create and organize word lists
- Spaced repetition review schedule
- Import/export (CSV/JSON)
- Desktop & mobile friendly UI

## Quick start
Requirements: Node.js (14+) and npm

Install and run:
```bash
git clone https://github.com/yourname/Vokab.git
cd Vokab
npm install
npm start
```

Build for production:
```bash
npm run build
```

## Development
- Follow conventional commits
- Run tests: `npm test`
- Lint: `npm run lint`
